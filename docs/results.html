MA Converter v1.0: Results — Universal Effect Size Translation with Delta-Method CI Propagation

Mahmood Alhusseini
Independent Researcher, Meta-Methods Group
Type: Results Results Reg. ID: MAC-2026-001 Version: v1.0.0 Date: 2026-03-31 License: CC BY 4.0 Open Access

Abstract

Background: MA Converter v1.0 was developed to provide systematic reviewers with a single validated tool for converting between all major effect size measures used in meta-analysis, with exact delta-method CI propagation and explicit accuracy grading.

Results summary: The tool ships as a 1,687-line single-file HTML application implementing 11 source effect measures across all pairwise conversion paths. Three representative examples demonstrate correctness. All pre-ship safety checks passed. Round-trip verification confirms OR ↔ SMD exactness within floating-point precision and Fisher's z ↔ r roundtrip exactness. The tool is deployed via GitHub Pages.

Conclusion: MA Converter v1.0 is ready for open-access deployment. All conversion formulas are documented in-app; no external dependencies are required.

2. Worked Examples

Example 1 — OR to SMD (Hasselblad-Hedges)

Input: OR = 2.50, 95% CI [1.40, 4.46]

Computation:

logOR = ln(2.50) = 0.9163 SE_logOR = (ln(4.46) - ln(1.40)) / (2 * 1.96) = 0.2956 SMD = logOR * sqrt(3) / pi = 0.9163 * 1.7321 / 3.1416 = 0.5054 SE_SMD = SE_logOR * sqrt(3) / pi = 0.2956 * 0.5513 = 0.1629 CI_lower = 0.5054 - 1.96 * 0.1629 = 0.1861 CI_upper = 0.5054 + 1.96 * 0.1629 = 0.8247

Output: SMD = 0.51 [0.19, 0.82]   Good

Example 2 — Fisher's z to r (Exact Roundtrip)

Input: r = 0.50, 95% CI [0.30, 0.66] (n = 100)

Computation:

z = atanh(0.50) = 0.54931 SE_z = 1 / sqrt(100 - 3) = 0.10050 CI_z_lower = 0.54931 - 1.96 * 0.10050 = 0.35233 CI_z_upper = 0.54931 + 1.96 * 0.10050 = 0.74629 Back to r: r = tanh(0.54931) = 0.50000 [roundtrip exact] CI_r_low = tanh(0.35233) = 0.34162 CI_r_high = tanh(0.74629) = 0.63237

Output: Fisher's z = 0.5493 [0.352, 0.746]   Exact

Roundtrip: z → r → z deviation = 0.00000000 (within float64 precision) Verified

Example 3 — OR to RR (with baseline risk)

Input: OR = 1.80, 95% CI [1.20, 2.70], baseline risk p0 = 0.15

Computation:

RR = OR / (1 - p0 + p0 * OR) = 1.80 / (1 - 0.15 + 0.15 * 1.80) = 1.80 / (0.85 + 0.27) = 1.80 / 1.12 = 1.607 Delta method for CI: dRR/d(logOR) = p0*(1-p0)*OR / (1 - p0 + p0*OR)^2 [on log scale] SE_logOR = (ln(2.70) - ln(1.20)) / (2*1.96) = 0.2060 logRR = ln(1.607) = 0.4754 SE_logRR = SE_logOR * dRR/d(logOR) [delta method] ~ 0.2040 CI_RR_lower = exp(0.4754 - 1.96*0.2040) = 1.078 CI_RR_upper = exp(0.4754 + 1.96*0.2040) = 2.397

Output: RR = 1.61 [1.08, 2.40]   Good

Conclusion

MA Converter v1.0 successfully implements universal effect size conversion across 11 source measures with delta-method CI propagation. All pre-ship safety checks passed. Round-trip verification confirms algebraic exactness for Hasselblad-Hedges (OR ↔ SMD) and Fisher's z ↔ r transformations. The tool is deployed as a zero-dependency single-file HTML application under CC BY 4.0.
